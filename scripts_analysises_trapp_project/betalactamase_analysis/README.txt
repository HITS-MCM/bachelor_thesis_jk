These files analyse the output for the betalactamase and contain all the data aswell as the scripts for the plots

r_analysis.qmd --> qmd-file that plotted the data in the plots used for bachelor thesis for lactamase
betalactamase.xlsx --> contains all the data used for the analysis as one excel file  
analysis_betalactamase.ipynb --> script that took all the data from the output of each analysis run  and were then stored in the betalactamase.xlsx file for further us

Be aware that some files will encounter some errors as the paths changed due to the migration into basement from the home directory


For the docking runs PocketVina was used. For further understanding how these were performed inquire the redmine directory of PocketVina
