Explanation of the scripts

scrps_docking --> contains the python scripts that are needed to dock all the ligands to one structure. It contains also the script "redocking" which is used to dock all the ligands totheir native crystall structure
data.xlsx --> contains the data summarized and which was used for the analysis of the hsp90 results
r_analysis_kbbox.qmd --> contains the qmd-file that generated all the plots used in the thesis 
analysis_kbbox_datase.ipynb --> script that gets all the data from the analysises to store them in one final excel sheet.

Be aware that paths in the scripts could not work due to the fact that all the files were stored in basement
